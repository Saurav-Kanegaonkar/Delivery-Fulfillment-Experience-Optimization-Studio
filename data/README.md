# Data Sources

Synthetic source-style data for digital delivery fulfillment, partner operations, order experience, and customer feedback optimization.

- `entities.csv`: 36 delivery journey records.
- `daily_metrics.csv`: 5,040 delivery journey-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
